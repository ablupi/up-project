## 项目下载

- 电脑上有git环境的可以直接拉下来使用，`git clone git@github.com:ablupi/up-project.git`
- 没有git环境就直接下载压缩包

## 项目运行

在项目根目录中打开终端（cmd），依次运行以下指令

- `npm install`
- `yarn dev`或者`npm run dev`
